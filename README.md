# Box Delivery - Гра на Unity

## Огляд проєкту

**Box Delivery** - це 2D аркадна гра з видом зверху, де гравець управляє працівником складу, що доставляє коробки до відповідних кольорових зон. Гра поєднує просте керування з елементами тайм-менеджменту та поступового ускладнення.

## Основні механіки

### Керування:
- **WASD** або **Стрілки** - рух персонажа у чотирьох напрямках
- **E** - підняти/опустити коробку
- **ESC** - пауза

### Геймплей:
1. Коробки з'являються в точках спавну
2. Гравець повинен підняти коробку (натиснути E)
3. Доставити коробку до зони з відповідним кольором
4. За правильну доставку +10 очок
5. За неправильну доставку -5 очок
6. Завдання: доставити 3 коробки до кінця часу

### Система винагород:
- Очки за доставку коробок
- Таймер під час гри
- Лічильник доставлених коробок
- Win/Lose умови

## Скрипти - Опис

### GameManager.cs
Управління основним ігровим циклом:
- Таймер рівня
- Підрахунок очок
- Лічильник доставлених коробок
- Логіка паузи (ESC)
- Умови перемоги/поразки

**Публічні методи:**
- `AddScore(int amount)` - додати очки
- `BoxDelivered()` - сигнал про доставку коробки
- `IsPaused()` - чи гра на паузі
- `IsGameEnded()` - чи гра закінчилась

### PlayerMovement.cs
Контроль руху персонажа:
- Читання введення WASD/стрілок
- Рух з константною швидкістю
- Контроль напрямку для анімації
- Підтримка паузи

### PlayerInteraction.cs
Взаємодія з коробками:
- Підняття коробок (E)
- Опускання коробок (E)
- Переміщення коробки разом з персонажем
- Область взаємодії (1.5 юнітів)

### BoxObject.cs
Логіка коробок:
- Колір коробки
- Статус переноски
- Оновлення візуального кольору

### DeliveryZone.cs
Логіка зон доставки:
- Детекція коробки в зоні
- Перевірка кольору
- Додавання очок
- Видалення коробки при доставці

### LevelManager.cs
Управління спавном коробок:
- Автоматичний спавн коробок
- Контроль інтервалу спавну
- Обмеження на кількість коробок одночасно
- Випадковий вибір кольору коробки

### CameraController.cs
Слідування камери:
- Плавне слідування за персонажем
- Налаштовуваною швидкістю

## Встановлення та Запуск

### Вимоги:
- Unity 2022.3 или вище
- TextMeshPro (зазвичай включено за замовчуванням)

### Кроки Встановлення:

1. **Клонуйте репозиторій** або відкрийте проєкт у Unity

2. **Відкрийте сцену**: Assets/Scenes/GameScene.unity

3. **Нажміть Play** - гра повинна запуститися

### Якщо це Перша Установка:

Дотримуйтесь SETUP_GUIDE.md для створення всіх необхідних об'єктів та префабів.

## Налаштування Гри

### У GameManager:
- `Level Time` - тривалість рівня в секундах (за замовчуванням: 60)
- `Boxes To Win` - кількість коробок для перемоги (за замовчуванням: 3)

### У LevelManager:
- `Spawn Interval` - час між спавном коробок (за замовчуванням: 2 сек)
- `Max Boxes At Once` - максимум коробок одночасно (за замовчуванням: 5)

### У PlayerMovement:
- `Speed` - швидкість руху персонажа (за замовчуванням: 5)

### У PlayerInteraction:
- `Interact Range` - радіус взаємодії з коробками (за замовчуванням: 1.5)

## Розширення Гри

### Додавання Нових Рівнів:
1. Дублюйте GameScene.unity
2. Змініть кількість коробок для перемоги
3. Змініть тривалість таймера
4. Додайте більше перешкод

### Додавання Анімацій:
1. Створіть анімацію персонажа у 4 напрямках
2. Додайте Animator компонент до персонажа
3. PlayerMovement автоматично встановлює параметри "MoveX", "MoveY", "IsMoving"

### Додавання Звуків:
1. Додайте AudioSource компоненти до об'єктів
2. Грайте звуки при доставці, підніманні коробок і т.д.

### Додавання Перешкод:
1. Створіть об'єкти зі спрайтом та Collider 2D
2. Встановіть is Trigger = false для фізичного перехоплення

## Кольори

Використовуються 5 кольорів для коробок та зон:
- **Red** (Червоний) - RGB(1, 0, 0)
- **Green** (Зелений) - RGB(0, 1, 0)
- **Blue** (Синій) - RGB(0, 0, 1)
- **Yellow** (Жовтий) - RGB(1, 1, 0)
- **Orange** (Оранжевий) - RGB(1, 0.5, 0)

## Можливі Проблеми та Рішення

### Коробки не спавняються:
- Перевірте, чи призначений Box префаб у LevelManager
- Переконайтесь, що spawn points створені і назначені

### Персонаж не рухається:
- Перевірте, чи Player має Rigidbody 2D компонент
- Переконайтесь, що PlayerMovement script додан

### Коробки не піднімаються:
- Перевірте, чи Player має PlayerInteraction компонент
- Переконайтесь, що carryPoint назначена

### Коробки не зникають при доставці:
- Перевірте, чи DeliveryZone має is Trigger = ON
- Переконайтесь, що зоне назначен правильний колір
