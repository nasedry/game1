# Box Delivery - Довідник Скриптів

## Огляд всіх Скриптів

### 1. **GameManager.cs** ⭐ КРИТИЧНО ВАЖЛИВО
**Мета**: Управління основним ігровим циклом, очками, таймером та станом гри

**Компоненти**:
- Синглтон (instance)
- Таймер гри
- Лічильник очок
- Лічильник доставлених коробок
- Система паузи (ESC)

**Публічні методи**:
```csharp
AddScore(int amount)      // Додати очки
BoxDelivered()            // Сигнал про доставку коробки
IsPaused()                // Чи гра на паузі
IsGameEnded()             // Чи гра закінчилась
RestartGame()             // Перезавантажити гру
TogglePause()             // Увімкнути/вимкнути паузу
```

**Налаштування**:
- `levelTime` - Тривалість рівня (сек)
- `boxesToWin` - Кількість коробок для перемоги

---

### 2. **PlayerMovement.cs**
**Мета**: Контроль руху персонажа за допомогою клавіатури

**Компоненти**:
- Читання введення (WASD/стрілки)
- Рух з постійною швидкістю
- Підтримка паузи
- Параметри анімації

**Публічні властивості**:
- `speed` - Швидкість руху персонажа (за умовчанням: 5)

**Параметри Animator**:
- `MoveX` (float) - Напрямок по X
- `MoveY` (float) - Напрямок по Y
- `IsMoving` (bool) - Чи рухається персонаж

---

### 3. **PlayerInteraction.cs**
**Мета**: Взаємодія з коробками (підняття/опускання)

**Компоненти**:
- Детекція коробок поблизу
- Підняття коробки (E)
- Опускання коробки (E)
- Переміщення коробки разом з персонажем
- Залишення коробки на місці при розв'язанні гри

**Публічні методи**:
```csharp
GetCarriedBox()    // Отримати поточну коробку
ClearCarriedBox()  // Очистити посилання на коробку
```

**Налаштування**:
- `carryPoint` - Точка утримання коробки (дочірній об'єкт)
- `interactRange` - Радіус взаємодії (за умовчанням: 1.5)

---

### 4. **BoxObject.cs**
**Мета**: Логіка коробок та їхніх властивостей

**Компоненти**:
- Колір коробки
- Статус переноски
- Оновлення візуального вигляду
- Система кольорів

**Публічні методи**:
```csharp
GetBoxColor()           // Отримати колір коробки
SetBoxColor(string color) // Встановити колір коробки
OnDelivered()           // Викликається при доставці (видаляє об'єкт)
```

**Доступні кольори**:
- "Red" (Червоний)
- "Green" (Зелений)
- "Blue" (Синій)
- "Yellow" (Жовтий)
- "Orange" (Оранжевий)

---

### 5. **DeliveryZone.cs**
**Мета**: Зони доставки коробок і перевірка правильності доставки

**Компоненти**:
- Детекція коробок в зоні
- Перевірка кольору коробки
- Додавання очок при правильній доставці
- Штраф за неправильну доставку

**Налаштування**:
- `zoneColor` - Колір зони (повинен відповідати коробкам)

**Публічні методи**:
```csharp
GetZoneColor()        // Отримати колір зони
SetZoneColor(string color) // Встановити колір зони
```

**Логіка**:
- Правильна доставка: +10 очок
- Неправильна доставка: -5 очок

---

### 6. **LevelManager.cs**
**Мета**: Управління спавном коробок протягом гри

**Компоненти**:
- Автоматичний спавн коробок
- Контроль інтервалу спавну
- Обмеження на кількість коробок
- Випадковий вибір кольору

**Публічні методи**:
```csharp
BoxRemoved()              // Зменшити лічильник коробок
SetSpawnInterval(float)   // Встановити інтервал спавну
SetMaxBoxes(int)          // Встановити макс. коробок одночасно
```

**Налаштування**:
- `boxPrefab` - Префаб коробки
- `spawnPoints` - Масив точок спавну
- `spawnInterval` - Інтервал між спавнами (сек)
- `maxBoxesAtOnce` - Максимум коробок одночасно

---

### 7. **CameraController.cs**
**Мета**: Плавне слідування камери за персонажем

**Компоненти**:
- Пошук персонажа автоматично
- Плавне рух камери
- Налаштовуваний offset

**Налаштування**:
- `player` - Посилання на персонажа
- `smoothSpeed` - Швидкість плавності (за умовчанням: 0.3)
- `offset` - Зміщення від персонажа

---

### 8. **DebugHelper.cs** (Опціонально)
**Мета**: Допоміжний скрипт для тестування та налагодження

**Гарячі клавіші**:
- `T` - Додати 10 очок
- `R` - Перезавантажити гру
- `D` - Вивести статус гри в консоль
- `B` - Сигнал про доставку коробки

---
